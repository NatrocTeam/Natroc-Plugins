# Technical Documentation Example

## User request

Write documentation for a CLI command that installs plugins.

## Weak AI-style answer

This command streamlines the plugin installation workflow by enabling users to efficiently install plugins from the marketplace.

## Human-style answer

Use this command to install one or more plugins from the marketplace.

```bash
awesome-agent install plugin-a plugin-b
```

The command searches the marketplace, checks the plugin metadata, and installs the selected plugins into the local agent configuration.

## Notes

Good technical documentation should:

- explain what the command does
- show the command
- avoid marketing language
- mention important behavior clearly
