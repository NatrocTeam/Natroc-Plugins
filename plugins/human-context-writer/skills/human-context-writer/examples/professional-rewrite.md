# Professional Rewrite Example

## User request

Rewrite this so it sounds more professional but not stiff:

"hey so basically this feature adds a way for users to reset their password if they forgot it, its pretty straightforward"

## Human-style rewrite

This feature adds a password reset flow for users who have lost access to their account. The implementation is straightforward and follows the existing authentication pattern.

## Notes

A good professional rewrite should:

- clean up casual language without making it sound corporate
- keep the original intent visible
- not add fake enthusiasm or dramatic language
- not change the actual meaning of what was said
