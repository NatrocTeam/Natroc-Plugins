# Product Copy Example

## User request

Write a short description for a CLI tool that helps developers find and install agent skills.

## Weak AI-style description

Awesome Agent CLI is a powerful and innovative tool that empowers developers to seamlessly discover, leverage, and install cutting-edge agent skills from a centralized marketplace, streamlining the development workflow and enhancing overall productivity.

## Human-style description

Awesome Agent CLI is a command-line tool for finding and installing agent skills from the marketplace.

No manual setup. Search for what you need, install it, and move on.

## Notes

Good product copy should:

- say what the product actually does in plain terms
- avoid words like "powerful", "seamless", "cutting-edge", "leverage", "empower", "streamline"
- be short enough to read without effort
- not end with a generic benefit like "enhancing overall productivity"
- let the product speak for itself
