# Social Post Example

## User request

Write a short post about AI and coding.

## Weak AI-style post

AI is transforming the way developers write code by improving productivity, enhancing workflows, and enabling innovation across the software development lifecycle.

## Human-style post

AI can write code fast. That part is obvious.

The harder part is knowing which code to accept.

If you cannot explain what the AI generated, you are not saving time. You are just moving the problem to later.

## Notes

A good social post should:

- have a clear point of view
- avoid generic motivational tone
- use short lines when useful
- say something specific
