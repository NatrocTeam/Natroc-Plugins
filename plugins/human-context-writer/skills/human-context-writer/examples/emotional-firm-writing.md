# Emotional and Firm Writing Example

## User request

Write a message telling the designer that we have already asked for the dark mode version three times and it still has not been delivered.

## Weak AI-style answer

I hope this message finds you well. I wanted to follow up regarding the dark mode version that we previously discussed. We have made several requests and would greatly appreciate an update on the current status at your earliest convenience.

## Human-style answer

We have asked for the dark mode version three times now and it still has not been delivered. Can you give us a clear update on when this will be ready? We need a specific date, not a general estimate.

## Notes

A good firm message should:

- state the facts clearly without dressing them up
- not open with "I hope this message finds you well"
- not soften the point with vague corporate language
- ask for something specific
- stay professional without being rude unless the user explicitly wants a stronger tone
