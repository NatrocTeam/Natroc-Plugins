# Email Writing Example

## User request

Write a professional email asking for the correct SVG file.

## Human-style email

Subject: Request for the SVG file

Hi,

Thanks for sending the file.

I checked it, but it looks like the file is still a PNG, not an actual SVG. Could you please send the original SVG file or the SVG code directly?

I need the SVG version so I can edit it and use it properly in the project.

Thank you.

## Notes

Good email writing should:

- be clear
- avoid sounding aggressive unless requested
- explain what is needed
- explain why it is needed
